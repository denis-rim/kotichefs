# KotiChefs - A course project for the course [Full Stack open 2021]
### https://fullstackopen.com/en/ 

## Features
1. Register a user
2. Verify user's email address
3. Send forgot password email
4. Reset password
5. Get current user info
6. Login
7. Logout
8. Access token
9. Refresh token
10. Get user's profile
11. Create a product
12. Get all products
13. Get a product by id
14. Update a product
15. Delete a product


## What you will need
* A running instance of MongoDB
* An IDE or text editor (VS Code)
* A web browser
* A package manager such as NPM or Yarn
* Node.js installed

## What technology?
Server:
- TypeScript 
- Express
- MongoDB with Mongoose
- Zod
- jsonwebtoken
- bcrypt
- Nodemailer
- Pino
- node-config
- node-mailer

Client:
- TypeScript
- React
- Next
- Axios
- Zod
- react-hooks-form
